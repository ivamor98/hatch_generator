# Генератор штриховки прямоугольника
## 1 Описание проекта
Данный файл содержит в себе модуль для генерации штриховки (hatch) внутри заданного контура квадратной или прямоугольной детали. Стороны квадрата параллельны осям.
На вход программа принимает массив точек, описывающий простой квадрат или прямоугольник,
угол наклона штриховки относительно горизонтальной оси в градусах (0–180°) и шаг между линиями штриховки в мм (положительное число).
На выходе печатаются координаты линий штриховки.
Программа запускается командой: ./hatch_generator --angle 45 --step 1
где --angle - угол наклона штриховки, --step - шаг.

**Неверный формат вызыва программы может привести к ее аварийному завершению**
*Проект выполнялся в учебных целях, и не имеет продуманной структуры.
